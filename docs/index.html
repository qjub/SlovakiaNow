Ako funguje SlovakiaNow

Python scraper beží automaticky každé 4 hodiny cez GitHub Actions. Sťahuje dáta z primárnych verejných zdrojov — EU inštitúcií, štátnych agentúr SR, RSS feedov médií.

Výstup sa ukladá ako docs/data/latest.json v GitHub repozitári. Frontend číta tento súbor priamo — bez servera, bez databázy.

Žiadna redakcia, žiadna agenda. Kód je open-source a overiteľný každým.
