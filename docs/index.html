Ako funguje SlovakiaNow

Architektúra bez servera. Všetky dáta sa načítavajú priamo z API pri každom otvorení stránky — browser volá ECB, Eurostat a Open-Meteo priamo (majú Access-Control-Allow-Origin: *).

PHM ceny (ŠÚSR) sú dostupné len cez HTML scrape — používame CORS proxy s automatickým fallback na zálohu.

Žiadna redakcia, žiadna agenda. Kód je open-source a overiteľný každým.

→ GitHub: qjub/SlovakiaNow
