🔍 Transparentnosť & Metodológia

Všetky dáta pochádzajú z primárnych verejných zdrojov. Python scraper beží automaticky každé 4 hodiny cez GitHub Actions a ukladá výsledok do docs/data/latest.json v tomto repozitári. Frontend číta tento súbor priamo — bez prostredníka, bez úprav. Celý kód je open-source a overiteľný.
